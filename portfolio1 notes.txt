how use poppins font without google link.
download poppins font.
go to https://transfonter.org/
add font poppins-regular.ttf -> convert to woff2
download
keep it in public/fonts/poppins-regular.woff2

use the font-face provided by transfonter, in css file or in style tag in head.
@font-face {
    font-family: 'Poppins';
    src: url('Poppins-Regular.woff2') format('woff2');
    font-weight: normal;
    font-style: normal;
    font-display: swap;
}
-
check if the font loaded: instect > network > font > refresh page
see if Poppins-Regular.woff2 loaded: status code: 200 ok
